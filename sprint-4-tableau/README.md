# Tableau Superstore Profitability & Advertising ROI Analysis

## Overview

This project analyzes Superstore sales, profitability, advertising performance, and product returns using Tableau dashboards and visual analytics.

The objective of the project was to identify:
- high-performing regions,
- profitable product categories,
- advertising opportunities,
- and the impact of returns on profitability.

This project demonstrates business intelligence and data visualization techniques using Tableau.

---

## Business Problem

Retail businesses need to understand which products, regions, and advertising strategies generate the highest profitability while minimizing losses from product returns.

This analysis helps identify:
- top-performing regions,
- low-profit subcategories,
- seasonal advertising opportunities,
- and return patterns affecting business performance.

---

## Tools & Technologies Used

- Tableau
- Data Visualization
- Calculated Fields
- Dashboard Filters
- Data Joins
- Business Analytics
